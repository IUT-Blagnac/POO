3. Mthodes coutTotal() et coutSuppl()

Les spcialistes donnent les spcification suivantes :

  1. Classe Vehicule
    1. Le cot total d’un Vehicule est de 15% de la distance parcourue fois la consommation.
  2. Classe Voiture
    1. Le cot total d’une Voiture est l’addition de :
      • cot total d’un vhicule,
      • un forfait de cot de 0.85 fois la distance parcourue,
      • le cot supplmentaire particulier aux voitures.
    2. Le cot supplmentaire particulier aux voitures est la distance parcourue multiplie par le taux d’usure (en pourcentage).
  3. Classe Camion
    1. Le cot total d’un Camion est l’addition de :
      • cot total d’un vhicule,
      • un forfait de 1.35 fois la distance parcourue,
      • le cot supplmentaire particulier aux camions.
    2. Le cot supplmentaire particulier aux camions est la distance parcourue multiplie par le ratio (en pourcentage) et la charge utile.

On souhaite faire fonctionner le code suivant :

Vehicule tabv[] = new Vehicule [4];

tabv[0] = new Voiture ("2222 VV 22", 5);
tabv[1] = new Camion ("2222 CC 22", 12);
tabv[2] = new Voiture ("3333 VV 33", 6, "Opel", 10);
tabv[3] = new Camion ("3333 CC 33", 14, 1000, 2);

for (int i=0 ; i<4 ; i++) {
        tabv[i].afficher();
        System.out.println (tabv[i].coutTotal());
        System.out.println (tabv[i].coutSuppl());
}

Questions :

  1. crire les mthodes coutTotal() et coutSuppl() dans les 3 classes.
  2. Nous n’avons pas de dfinition pour le cot supplmentaire particulier aux vhicules. Il n’existe pas. Doit-on dclarer la mthode coutSuppl() dans la classe Vehicule ? Comment l'crire ?
Solution

crire coutSuppl() dans la classe Vehicule :

  1. Ncessaire pour que le code de test marche : System.out.println (tabv[i].coutSuppl());
  2. On ne peut "que" mettre 0. 
class Vehicule
{
        // ...
   public double coutTotal ()  {
       return this.distParcourue * this.consommation * 0.15;
   }
   public double coutSuppl () {
       return 0;
   }
}

class Voiture extends Vehicule
{
        // ...
        public double coutTotal ()  {
                return super.coutTotal() + this.coutSuppl()
                                + this.getKilometrage() * 0.85;
        }
        public double coutSuppl ()  {
                return this.getKilometrage() * this.tauxUsure/100 ;
        }
} // Fin Classe Voiture

class Camion extends Vehicule
{
        // ...

        public double coutTotal ()      {
                return super.coutTotal() + this.coutSuppl()
                        + this.getKilometrage() * 1.35 ;
        }
        public double coutSuppl ()      {
                return this.getKilometrage () * this.ratio / 100 ;
        }
} // Fin Classe Camion
