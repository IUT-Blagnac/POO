3. Test du polymorphisme

3.1. Préparation : ajout de méthodes

Questions :

  1. Dans la classe Compte, déclarer une nouvelle méthode :

    1. public void afficherAvecSolde ().

    2. Elle affiche le Compte (méthode this.afficher()) et ajoute le solde à l'écran (this.soldeCompte()).

  2. Dans la classe Compte, déclarer une nouvelle méthode :

    1. public void transferer (Compte pfC, double pfSomme) throws CompteException.

    2. Elle transfère (virement) pfSomme euros de l’objet courant (retrait, this.retirer(pfSomme)) vers le compte pfC (dépôt, pfC.deposer(pfSomme)).

    3. Elle lève une exception CompteException

      1. Si le Compte pfC vaut null.

      2. Si le dépôt ou le retrait sur un des deux comptes ne peut pas avoir lieu normalement. En fait elle "laisse" passer les exceptions levées par retirer() ou deposer().
