1. Test ArrayList<E> : utiliser une ArrayList<String>

Nous allons tester la classe ArrayList<E> : nous allons créer une ArrayList d’objets String.

La documentation de la classe ArrayList est accessible depuis documentation java et se trouve ici : documentation ArrayList<E>.

Pour démarrer : créer un nouveau programme java :

  1. Lancer Scite.

  2. Créer un classe, par exemple TP4Exo1.

  3. Enregistrer le fichier avec le bon nom de fichier.

  4. Ajouter la méthode main().

  5. Ca doit compiler (afficher "Hello World" pour tester) …

En parallèle de la lecture de la documentation et/ou du support de cours, on testera les différentes méthodes de la classe ArrayList<E> :

  1. Déclarer une variable alChaines de type ArrayList de String.

    NB : la classe ArrayList appartient au package java.util.

  2. Créer avec alChaines une ArrayList de String avec le constructeur par défaut.

  3. Faire quelques ajouts de chaînes de caractères en fin de alChaines avec la méthode add() (cf. support de cours).

  4. Afficher la taille (ou nombre d’éléments contenus) de alChaines (cf. support de cours).

  5. Par une boucle : afficher les éléments de alChaines (cf. support de cours, méthodes get(), size()).

  6. Faire quelques ajouts en fin, au début, au milieu. Regarder les méthodes et surcharges associées de add() (cf. support de cours). Après chaque ajout, afficher la taille de alChaines.

  7. Par une boucle : afficher les éléments de alChaines. Afficher la taille de alChaines.

  8. Faire quelques ajouts de plus par un for (15-25) (tester alChaines.add("chaine "+i)).

  9. Afficher les éléments de alChaines et sa taille.

  10. Modifier quelques éléments contenus dans alChaines : le premier, le dernier, quelques uns au milieu. Ne pas faire retrait/ajout, mais utiliser la méthode set() (cf. support de cours).

  11. Afficher les éléments de alChaines et sa taille.

  12. Tester la méthode toString() sur alChaines.

  13. Faire quelques suppressions : premier élément, dernier élément, au milieu. (cf. support de cours). Après chaque suppression, afficher la taille (nombre d’éléments contenus) de alChaines.

  14. Vider alChaines en une instruction.

  15. Afficher les éléments de alChaines et sa taille.

  16. Étudier la documentation de la méthode trimToSize(). A quoi sert-elle ? (Eventuellement étudier la méthode … et afficher son résultat avant ET après trimToSize()).

2. Manipuler une ArrayList<Compte>

Reprendre le programme suivant ET récupérer les packages du TP précédent (tps.banque, …).

Le programme principal :

  • remplit une ArrayList<Compte>,

  • affiche les comptes

  • réalise en boucle :

    • demande un nom de propriétaire

    • appelle la méthode getComptesDe() pour retrouver les comptes de ce propriétaire,

A faire : écrire le code de la méthode :

public static ArrayList<Compte> getComptesDe (String pfNomProprietaire, ArrayList<Compte> pfALComptes)

Solution recherche dans une ArrayList<Compte>
        /**
         * Recherche les comptes du propriétaire pfNomProprietaire
         * dans la liste de comptes pfALComptes.
         * @param pfNomProprietaire     Propriétaire recherché
         * @param pfALComptes   Liste dans laquelle les comptes sont recherchés
         * @return      Liste des compte de pALComptes ayant pour propriétaire pNomProprietaire. Liste vide si aucun compte trouvé.
         */
        public static ArrayList<Compte> getComptesDe (String pfNomProprietaire, ArrayList<Compte> pfALComptes) {
                int i, taille;
                ArrayList<Compte> result;

                result = new ArrayList<Compte>();
                taille = pfALComptes.size();
                for (i=0; i<taille; i++) {
                        if (pfALComptes.get(i).getProprietaire().equals(pfNomProprietaire)) {
                                result.add(pfALComptes.get(i));
                        }
                }

                return result;
        }
